Installing dependencies:
------------------------
1. distutils
sudo apt-get install python-distutils

2. pip
Download from https://pypi.python.org/pypi/pip/

3. pip install numpy
4. pip install scipy
5. pip install pandas
6. pip install theano
7. pip install nose
8. pip install matplotlib
9. pip install scikit-learn

10. Download qit library from:
http://sourceforge.net/projects/qit/
svn checkout svn://svn.code.sf.net/p/qit/code/python/trunk qit
Install qit by navigating to python/trunk and type
python setup.py install

Installing palm
---------------
After dependencies have been installed, run this command in the
the top directory:
python setup.py install

Tests are run by typing this command within the top directory:
nosetests
